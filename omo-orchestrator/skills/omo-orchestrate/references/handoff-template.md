# Handoff Template

Use this template when work spans agents, sessions, or context windows.

```text
Goal:

Current state:

Files inspected:

Files changed:

Tests and validation:

Review decision:

Blocking findings:

Verified non-issues:

Final approval state:

Open blockers:

Next exact action:

Do not touch:
```

Keep handoffs factual. Do not include secrets, tokens, private transcripts, or unrelated context.
